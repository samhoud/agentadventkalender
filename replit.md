# Christmas Tree Advent Calendar

## Overview
A festive interactive Christmas advent calendar with password protection. Built as a pure client-side web application using HTML, CSS, and vanilla JavaScript.

## Project Structure
- `index.html` - Main HTML file with password protection overlay and modal system
- `script.js` - Interactive functionality including password validation, modal controls, carousel, and special features
- `style.css` - Main styling including Christmas theme, animations, and responsive design
- `sleigh.css` - Sleigh animation styles
- `images/` - Image assets for the advent calendar

## Features
- Password protection (password: kerst2025)
- Interactive Christmas tree layout with ornaments
- Daily unlockable content
- Image carousel for special days
- Microsoft Forms integration for voting
- Random compliment generator
- Flip card animations
- Snow and background animations
- Flying sleigh animation on star click

## Technology Stack
- Pure HTML/CSS/JavaScript (no frameworks)
- Client-side only (static site)
- No build process required

## Setup
This is a static website that can be served directly with any HTTP server.
